# 下载列表

Scorpio （当前版本 v{{site.currently}}）提供以下 Nuget 包，请根据需要下载。

## Nuget 安装脚本

您可以通过 Nuget 服务器快速安装 Scorpio 到您的项目中。

``` bash
$ dotnet add package Scropio -v {{site.currently}}
```

``` powershell
PM> install-Package Scropio -v {{site.currently}}
```
